偶然接触到Tableau，正好最近对数据分析比较有兴趣加上学生身份可以免费获得Tableau Desktop + Pro有效期一年的使用权限，所以记录一下Tableau的学习心得和使用感受。主要侧重数据分析及可视化，前期的数据收集和清洗就不过多赘述。

什么是Tableau

Tableau（Tableau Software）是一款专业的数据分析与可视化工具，作为BI tool leader，它不仅支持连接本地或云端的各类数据，包括电子表格、数据库元数据，定时自动从元数据更新报表，拥有强大的数据处理能力，还能实现网页、手机、tablet间跨平台的可视化。

相对于Excel，Tableau属于偏小众的软件，相关的资料和教程也少很多。入门教程可以参考官网提供的教学白皮书和视频，以及coursera上的教程。

新手入门

  1. 连接数据：Tableau可以连接excel、csv以及mysql等各种数据库
  2. 度量和维度：度量是指数据表中的数值数据，而维度则是类别数据
  3. 选择合适的图表：tableau提供柱状图、点图、线图、饼图、直方图、地图等多种图表形式用于展示数据
